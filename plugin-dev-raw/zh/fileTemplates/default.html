在檔案模板中，您可以使用文本、程式碼、註釋和預定義變數。 以下提供了預定義變數的列表。 當您在模板中使用這些變數時，它們稍後會在編輯器中擴展為相應的值。

您也可以指定自訂變數。 自訂變數使用以下格式：${VARIABLE_NAME}，其中 VARIABLE_NAME是變數的名稱（例如，${MY_CUSTOM_FUNCTION_NAME}） 。 在 IDE 建立包含自訂變數的新檔案之前，您會看到一個對話框，您可以在其中定義模板中自訂變數的值。

通過使用 #parse 指令，可以包括 包含 頁籤中的模板。 要包含模板，請在引號中將模板的全名指定為參數（例如，#parse("File Header")）。
預定義變數列表
${DATE}   當前系統日期
${DAY}   當前月份的日期
${DAY_NAME_SHORT}   當前日期名稱的前 3 個字母（例如，Mon、Tue 等）
${DAY_NAME_FULL}   當前日期的全名（Monday、Tuesday 等）
${DIR_PATH}   新檔案的目錄路徑（相對於專案根目錄）
${DS}   美元符號 （$）。 此變數用於轉義美元字元，因此不會將其視為模板變數的前綴。
${FILE_NAME}   新檔名
${HOUR}   當前小時
${MINUTE}   當前分鐘
${SECOND}   當前秒
${MONTH}   當前月份
${MONTH_NAME_SHORT}   當前月份名稱的前 3 個字母（Jan、Feb 等）。
${MONTH_NAME_FULL}   當前月份的全名（January、February 等）。
${NAME}   新實體的名稱（檔案、類型、接口等）
${ORGANIZATION_NAME}   在專案設定中指定的組織名稱
${PRODUCT_NAME}   IDE 名稱
${PROJECT_NAME}   當前專案名稱
${TIME}   當前系統時間
${USER}   當前使用者的系統登入名
${YEAR}   當前年度
