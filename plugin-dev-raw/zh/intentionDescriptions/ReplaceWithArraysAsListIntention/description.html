將 Collections.emptyList() 或 Collections.singletonList() 呼叫取代為對 Arrays.asList() 的呼叫。

當專案或模組使用語言級別 9 時：

  • 該意圖將 Collections.emptyList() 或 Collections.singletonList() 取代為對 java.util.List.of() 的呼叫。
  • 將 Collections.emptySet() 或 Collections.singleton() 呼叫取代為 java.util.Set.of()。
  • 將 Collections.emptyMap() 或 Collections.singletonMap() 呼叫取代為 java.util.Map.of()。

當專案或模組使用 Guava 庫且語言級別為 9 或更低時，該意圖會將 Collections.emptyList() 或 Collections.singletonList() 呼叫取代為 com.google.common.collect.ImmutableList、com.google.common.collect.ImmutableSet 或 com.google.common.collect.ImmutableMap。
