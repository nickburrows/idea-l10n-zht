報告不以呼叫 "super" 建構函式或同一類的另一個建構函式開頭的建構函式。

可以認為是此類別建構函式隱式地從呼叫 super() 開始。 某些編碼標準傾向於顯式地呼叫 super()。

範例：


  class Foo {
    Foo() {}
  }

在套用快速修復後：


  class Foo {
    Foo() {
      super();
    }
  }

使用檢查設定可以忽略直接從 Object 擴展的類別。 例如： 


  class Foo {
    Foo() {} // 未報告
  }

  class Bar extends Foo {
    Bar() {} // 對 'super()' 的隱式呼叫
  }
