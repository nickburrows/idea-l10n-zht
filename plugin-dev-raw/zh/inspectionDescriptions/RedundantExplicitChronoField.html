當將 java.time.temporal.ChronoField 和 java.time.temporal.ChronoUnit 作為實參的 java.time 方法呼叫可被取代為更具體的方法時，報告這些呼叫。

範例：


LocalTime localTime = LocalTime.now();
int minute = localTime.get(ChronoField.MINUTE_OF_HOUR);

在套用快速修復後：


LocalTime localTime = LocalTime.now();
int minute = localTime.getMinute();
