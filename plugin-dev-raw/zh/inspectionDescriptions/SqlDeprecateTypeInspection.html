報告已棄用並且可能在 DBMS 的未來版本中消失的類型的用法。

報告的類型：

  • Oracle 中的 LONG（請參閱 docs.oracle.com 上的 Deprecated and Desupported Features 頁面）。
  • Microsoft SQL Server 中的 TEXT、NTEXT 和 IMAGE（請參閱 docs.microsoft.com 上的 Deprecated Database Engine Features in SQL Server 2016 頁面）。

範例（Oracle）：

CREATE TABLE ot.foo(
a NUMBER GENERATED BY DEFAULT AS IDENTITY,
b LONG NOT NULL
);
