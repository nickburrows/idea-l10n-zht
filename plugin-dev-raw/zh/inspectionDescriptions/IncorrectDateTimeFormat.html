報告錯誤的日期時間格式模式。

報告以下錯誤：

  • 不支持的模式字母，如 "TT"
  • 使用保留字元，如 "#"
  • 不正確的填充用法
  • 不平衡的中括號
  • 連續模式字母的數量不正確

範例：


  DateTimeFormatter.ofPattern("[][]]"); // 右 ']' 之前沒有左 '['
  DateTimeFormatter.ofPattern("TT"); // 非法的模式字母 'T'
  DateTimeFormatter.ofPattern("{"); // 使用了保留字元 '{'
  DateTimeFormatter.ofPattern("MMMMMM"); // 過多的連續模式字母 'M'
