報告對超類別中從子類別建構函式中宣告的欄位的賦值或修改。

最好在其自己的建構函式中初始化超類別的欄位，並在子類別中委託給該建構函式。 如果它在建構後沒有改變，這還將允許宣告一個欄位 final。

範例：


  class Super {
    int x;
  }
  class Sub extends Super {
    Sub(int _x) {
      // Warning: x is declared in a superclass
      x = _x;
    }
  }

要避免此問題，請宣告超類別建構函式：


  class Super {
    final int x;

    Super(int _x) {
      x = _x;
    }
  }
  class Sub extends Super {
    Sub(int _x) {
      super(_x);
    }
  }
