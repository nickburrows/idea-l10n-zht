報告以不正確的方式處理 ProcessCanceledException 的情況。

不能以任何方式捕獲、吞下、記錄或處理 ProcessCanceledException 及其繼承者。 相反，必須將其重新拋出，以便基礎架構能夠對其進行正確處理。

檢查會報告顯式的 ProcessCanceledException 或其繼承者的捕獲，以及對覆蓋 ProcessCanceledException 的 RuntimeException、Exception 和 Throwable 的捕獲。

範例：


// bad:
try {
  // ...
} catch (ProcessCanceledException e) { // 不應吞下異常
}

// 錯誤:
try {
  // ...
} catch (ProcessCanceledException e) {
  LOG.error("Error occurred", e); // 不應記錄異常
  throw e;
}


// 優良:
try {
  // ...
} catch (ProcessCanceledException e) {
  // 其他動作
  throw e;
}
