報告在例程呼叫中不帶名稱使用的實參。 預設情況下，此檢查處於停用狀態。

有關命名參數和未命名參數之間差異的詳細資訊，請參閱 docs.microsoft.com 上的 Binding Parameters by Name (Named Parameters) 。

範例（Microsoft SQL 伺服器）：

CREATE FUNCTION foo(n INT, m INT) RETURNS INT AS
BEGIN
    RETURN n + m;
END;

CREATE PROCEDURE test AS
BEGIN
    foo n = 1, m = 2;

--- 以下呼叫缺少參數名稱，會被醒目提示
    foo 1, 2;
END;

foo 1, 2; 呼叫中的參數 1, 2 突出顯示，因為它們缺少名稱。
