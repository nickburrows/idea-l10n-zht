報告不正確的 @Profile 表達式：

  • Spring 組態檔案不得為空
  • 在 Spring 組態檔案表達式中，'&' 和 '|' 運算符不得在沒有圓括號的情況下混用

範例：


  // 組態正確
  @Configuration
  @Profile("a & (b | c)")
  public class MyConfiguration {...}

  // 空 
  @Configuration
  @Profile() // 報告“組態檔案表達式必須包含文本”
  public class MyConfiguration {...}

  // 不帶圓括號的混合運算符 
  @Configuration
  @Profile("a & b | c") // 報告“格式不正確的組態檔案表達式”
  public class MyConfiguration {...}
