報告可被分別取代為 Files.newInputStream() 或 Files.newOutputStream() 呼叫的 new FileInputStream() 或 new FileOutputStream() 表達式。
使用 Files 方法建立的流通常比使用流建構函式建立的流更有效。

範例：


  InputStream is = new BufferedInputStream(new FileInputStream(file));

在套用快速修復後：


  InputStream is = new BufferedInputStream(Files.newInputStream(file.toPath()));

如果語言級別為 10 或更高，則此檢查不會顯示警告，但快速修復仍然可用。

僅當專案或模組的語言級別為 7 或更高時，此檢查才會報告。
