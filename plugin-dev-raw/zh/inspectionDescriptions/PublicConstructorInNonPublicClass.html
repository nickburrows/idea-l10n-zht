報告非 public 類別中的所有 public 建構函式。

在存取級別較低的類別中，通常沒有理由建立 public 建構函式。 但請注意，此檢查會改變某些反射呼叫的行為。 特別是，Class.getConstructor() 將無法找到更新的建構函式（應改為使用 Class.getDeclaredConstructor()）。 如果您的程式碼或某些已使用的框架的程式碼通過 getConstructor() 依賴於建構函式的可存取性，請不要使用該檢查。

範例：


  class House {
    public House() {}
  }

在套用快速修復後：


  class House {
    House() {}
  }
