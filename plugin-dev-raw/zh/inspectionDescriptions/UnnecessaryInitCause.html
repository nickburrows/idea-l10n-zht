報告對 Throwable.initCause() 的呼叫，其中異常建構函式還包含 Throwable cause 實參。

在本例中，可以移除 initCause() 呼叫，並將其實參新增到對異常建構函式的呼叫中。

範例：


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing");
    wrapper.initCause(ex); // 不必要呼叫 'Throwable.initCause()'
    throw wrapper;
  }

可通過快速修復將 cause 實參傳遞給建構函式。 在套用快速修復後：


  try {
      process();
  }
  catch (RuntimeException ex) {
    RuntimeException wrapper = new RuntimeException("Error while processing", ex);
    throw wrapper;
  }
