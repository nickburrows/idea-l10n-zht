報告可能縮小其存取修飾符範圍的欄位、方法或類別。

範例：


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // 可以為 private
  }

在套用快速修復後：


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

使用檢查選項可以定義修飾符更改建議的規則。
