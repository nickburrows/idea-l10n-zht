報告超過定義的字元長度的變數。

範例（Microsoft SQL 伺服器）：

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(5) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;

myVarExceeded 變數被定義為 VARCHAR(5) 但兩個分配的值（'abcde12345' 和 '123456789') 都超出了此限制。 您可以截斷指定的值或增加定義的長度。 要增加長度，請使用增加類型長度 快速修復。

在套用快速修復後：

CREATE PROCEDURE test() AS
BEGIN
DECLARE myVarOk VARCHAR(5) = 'abcde';
DECLARE myVarExceeded VARCHAR(10) = 'abcde12345';

SET myVarOk = 'xyz';
SET myVarExceeded = '123456789';
END;
