報告可以取代為方法參照的 lambda。 方法參照比 lambda 更容易理解和閱讀，儘管它們通常取決於您的喜好。

範例：


  Runnable r = () -> System.out.println();

在套用快速修復後：


  Runnable r = System.out::println;

即使 lambda 不呼叫任何方法，該檢查也可能會建議方法參照，例如將 obj -> obj != null 取代為 Objects::nonNull。 使用設定 | 編輯器 | 程式碼樣式 | Java | 程式碼生成設定來組態特殊方法參照。
