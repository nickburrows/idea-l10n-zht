報告動態類型接收器上的 asDynamic 函式用法。

asDynamic 函式對動態類型的表達式無影響。

動態類型接收器上的 asDynamic 函式會導致執行時問題，因為 asDynamic 將在 JavaScript 環境中執行，而且此類別函式在執行時可能並不顯示。 預期方式是在常用 Kotlin 類型上使用此函式。

移除 "asDynamic" 呼叫快速修復可用於自動修正程式碼。

範例：


  fun wrongUsage(d: Dynamic) {
     d.asDynamic().foo() // <== 冗餘，快速修復會將呼叫表達式簡化為 "d.foo()"
  }
