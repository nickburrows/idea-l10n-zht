通過類別實例而不是類別本身報告對 static 方法和欄位的參照。

雖然 PHP 語言規範允許通過實例變數參照 static 成員，但讀者可能會誤認為該方法依賴於實例，從而使程式碼令人困惑。

該快速修復會將實例變數取代為類別名。

範例：


  class StaticClass {
    public static function someStaticFunction(): int
    {
        return 3;
    }

  }

  $var = new StaticClass
  $var::someStaticFunction();

在套用快速修復後：


  StaticClass::someStaticFunction()
