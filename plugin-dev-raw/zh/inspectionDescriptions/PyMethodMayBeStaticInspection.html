報告任何無需建立類別實例並且可以設定為 static 的方法。

範例：


class MyClass(object):
    def my_method(self, x):
        print(x)

如果套用從方法中生成函式快速修復，程式碼將變為：


def my_method(x):
    print(x)


class MyClass(object):
    pass

如果選擇將方法設為 static 快速修復，將新增 @staticmethod 裝飾器：


class MyClass(object):
    @staticmethod
    def my_method(x):
        print(x)
