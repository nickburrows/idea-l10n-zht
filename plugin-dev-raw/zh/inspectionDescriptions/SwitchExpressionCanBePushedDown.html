報告每個分支都有通用子表達式，因此 switch 可以移動到內部的 switch 表達式和語句。 此動作會縮短程式碼。 在許多情況下，將得到的 switch 表達式提取到單獨的變數或方法是合理的。

範例：


  switch (value) {
    case 0 -> System.out.println("zero");
    case 1 -> System.out.println("one");
    case 2, 3, 4 -> System.out.println("few");
    default -> System.out.println("many");
  }

在套用快速修復後：


  System.out.println(switch (value) {
    case 0 -> "zero";
    case 1 -> "one";
    case 2, 3, 4 -> "few";
    default -> "many";
  });

此檢查僅適用於採用箭頭語法的增強 switch。
