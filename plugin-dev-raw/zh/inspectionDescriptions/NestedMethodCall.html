報告用作另一個方法呼叫的參數的方法呼叫。

該快速修復會引入一個變數，使程式碼更簡單，也更易於偵錯。

範例：


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int x = f(y());
  }

在套用快速修復後：


  public int y() { return 1; }
  public int f(int x) { return 2 * x; }

  public void foo() {
      int y = y();
      int x = f(y);
  }

使用檢查選項可以切換以下各項的報告：

  • 欄位初始設定式中的方法呼叫
  • 對 static 方法的呼叫
  • 對簡單 getter 的呼叫
