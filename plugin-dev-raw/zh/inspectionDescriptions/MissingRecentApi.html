報告在比 <idea-version> @since-build (plugin.xml) 中指定的版本更新的版本中引入的 IntelliJ Platform API 的用法。

使用此類別 API 可能導致延伸模組與舊的 IDE 版本不相容。

為了避免在舊 IDE 版本中執行延伸模組時可能出現的問題，請相應地提高 since-build，或移除此 API 的用法。

請參閱 IntelliJ 平台延伸模組 SDK 文檔中的構建版本範圍，了解更多詳細資訊。

組態檢查：

如果 <idea-version> @since/until-build 特性在 plugin.xml 中未指定，請顯式設定 Since/Until。
