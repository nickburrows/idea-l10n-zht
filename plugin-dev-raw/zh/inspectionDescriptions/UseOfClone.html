報告 clone() 方法的實作、對它的呼叫，以及 java.lang.Cloneable 接口的使用。

某些編碼標準禁止使用 clone()，建議改用複製建構函式或 static 工廠方法。

該檢查會忽略在陣列上對 clone() 的呼叫，因為這是一種正確而又緊湊的陣列複製方式。

範例：


  class Copy implements Cloneable /*警告*/ {

    public Copy clone() /*警告*/ {
      try {
        return (Copy) super.clone(); // 警告
      } catch (CloneNotSupportedException e) {
        throw new AssertionError();
      }
    }
  }
