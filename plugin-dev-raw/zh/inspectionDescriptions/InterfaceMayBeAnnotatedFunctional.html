報告可以用 @FunctionalInterface 註解的接口（自 JDK 1.8 起提供）。

使用 @FunctionalInterface 對接口進行註解會指示該接口為函式式接口，不能再向其新增其他 abstract 方法。

範例：


  interface FileProcessor {
    void execute(File file);
  }

在套用快速修復後：


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }

僅當專案或模組的語言級別為 8 或更高時，此檢查才會報告。
