報告其中的 default 分支位於另一個 case 前面的 switch 語句或表達式。 此類別建構會造成不必要的困惑。 如果可能，可以通過快速修復將 default 分支移到最後位置。

範例：


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

在套用快速修復後：


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
