建議將使用賦值的欄位的初始化取代為欄位宣告中的初始化。

僅當欄位賦值位於實例或 static 初始設定式中時報告，並且將其與欄位宣告聯接可能比較安全。 在其他情況下（比如建構函式中的賦值），提供的快速修復不會醒目提示，因為修正可能會更改語意。

範例：


  class MyClass {
    static final int intConstant;
    
    static {
      intConstant = 10;
    }
  }
該快速修復會將賦予的值移至欄位初始設定式，並移除類別初始設定式（如果可能）： 

  class MyClass {
    static final int intConstant = 10;
  }
