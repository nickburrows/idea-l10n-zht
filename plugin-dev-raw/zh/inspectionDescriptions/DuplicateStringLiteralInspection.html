報告整個專案中在不更改的情況下複製的字串文字。 提供了兩個快速修復。 一個是為重複的字串引入一個常數並在整個專案中使用它，另一個是顯示特定字串文字的所有重複項的位置。

範例：


  class C1 { String CONST1 = "duplicate string"; }
  class C2 { String CONST2 = "duplicate string"; }

組態檢查：

  • 使用最小字串長度欄位設定檢測重複項所需的最小字串長度。
  • 使用忽略@PropertyKey 表達式選項可忽略作為實參傳遞給使用 org.jetbrains.annotations.PropertyKey 註解的方法的字串。
