報告區域變數宣告和存取器，以記錄增強型 'for' 語句（這些語句通常更緊湊）中可被取代為模式變數的組件。

範例：


record Record(Integer x, String y) {
}

public static void test(List<Record> records) {
    for (Record record : records) {
        System.out.println(record.y());
        Integer x = record.x;
        System.out.println(x);
    }
}

可以取代為：


record Record(Integer x, String y) {
}

public static void test(List<Record> records) {
    for (Record(Integer x, String y) : records) {
        System.out.println(y);
        System.out.println(x);
    }
}

僅當專案或模組的語言級別為 20 或更高時，此檢查才會報告。

  • 使用嵌套深度限制選項指定要報告的最大嵌套解構模式數
  • 使用要解構的記錄組件的最大數量選項指定要在解構模式中使用的記錄可以包含的組件的最大數量
  • 使用未使用的記錄組件的最大數量選項指定 for 語句中未使用的組件的最大數量
