報告對 testing.T.FailNow 和位於測試檔案 goroutine 中的類似方法的呼叫。

像 FailNow 這種方法會呼叫 runtime.Goexit，並停止呼叫 goroutine，而不是測試。 因此，必須僅從執行測試或基準的 goroutine 中呼叫。

有關 FailNow 的更多資訊，請參閱 go.dev 上的 func (*T) FailNow。

範例：

func TestFoo(t *testing.T) {
  go func() {
    t.Fatal("oops") // 退出 goroutine，而不是 TestFoo
  }()
}

在套用取代為 'Error' 和 'return' 快速修復後：

func TestFoo(t *testing.T) {
  go func() {
    t.Error("oops")
    return
  }()
}
