報告空類別和沒有任何定義類別的 Java 檔案。

如果一個類別不包含任何欄位、方法、建構函式或初始設定式，則為空類別。 在進行重大更改或重構後，空類別通常會保留下來。

組態檢查：

  • 使用有註解時忽略選項可以指定特殊註解。 該檢查將忽略用這些註解標記的類別。

  • 使用如果類別是父類型的參數化，則將其忽略選項，可以忽略參數化超類別的類別。 範例： 

    class MyList extends ArrayList<String> {}
  • 使用忽略 java.lang.Throwable 的子類別，可以忽略擴展 java.lang.Throwable 的類別。
  • 使用註釋計為內容選項可忽略包含註釋的類別。
