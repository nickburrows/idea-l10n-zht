報告其中一個實參為空字串的字串串聯。 此類別串聯不必要。 有時，會作為習慣用法將非 String 物件或基元轉換為 String，但在這種情況下，使用類似 String.valueOf 的方法會更清晰。

建議通過快速修復來簡化串聯。

範例：


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

在套用快速修復後：


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

使用僅報告可以刪除空字串而不進行其他更改的情況選項僅報告可刪除空字串而不使用 String.valueOf 轉換其他動作數的情況。
