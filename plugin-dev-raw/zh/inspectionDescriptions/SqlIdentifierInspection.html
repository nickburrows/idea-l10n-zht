報告在查詢中使用 SQL 預留關鍵字作為關鍵字名稱時的情況。

範例（Microsoft SQL 伺服器）：

CREATE TABLE select (identity INT IDENTITY NOT NULL, order INT NOT NULL);

我們使用 select、identity 和order 作為表名和列名。 但它們也是 Microsoft SQL Server 中的保留關鍵字。 因此，為了將它們用作查詢中的物件名稱，您必須用引號參照這些關鍵字。 要用引號參照它們，您可以使用為關鍵字加引號 快速修復。

在套用快速修復後：

CREATE TABLE [select] ([identity] INT IDENTITY NOT NULL, [order] INT NOT NULL);
