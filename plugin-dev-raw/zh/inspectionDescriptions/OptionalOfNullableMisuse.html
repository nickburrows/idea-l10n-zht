報告傳遞始終為 null 或始終為非 null 實參的 Optional.ofNullable() 用法。 在這種情況下使用 Optional.ofNullable() 沒有意義：Optional.empty() 或 Optional.of() 應分別用於顯式宣告建立始終為空或始終非空的可選項的意圖。 也可能在 Optional.ofNullable() 實參中存在錯誤，因此應對其進行檢查。

範例： 


Optional<String> empty = Optional.ofNullable(null); // 應為 Optional.empty();
Optional<String> present = Optional.ofNullable("value"); // 應為 Optional.of("value");
