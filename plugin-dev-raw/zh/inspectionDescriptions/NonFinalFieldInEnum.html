報告枚舉類型中的非 final 欄位。 非 final 欄位會引入全域可變狀態，這通常會被認為不可取。

範例：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

在套用快速修復後：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
使用“忽略不能設為 'final' 的欄位”選項可以僅對可使用快速修復設為 final 的欄位發出警告。
