報告類別的 implements 列表或接口的 extends 列表中已經由超類別實作或由超接口擴展的接口。 此類別宣告不必要，並且可以安全移除。

範例：


  class X implements One, Two {
  }
  interface One {}
  interface Two extends One {}

在套用快速修復後：


  class X implements Two {
  }
  interface One {}
  interface Two extends One {}

使用該選項可以不報告 extends 或 implements 列表中的 Serializable 或 Externalizable。
