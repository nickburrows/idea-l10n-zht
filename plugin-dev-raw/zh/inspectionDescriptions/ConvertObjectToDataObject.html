報告可轉換為 data object 的 object

data object 會自動生成 toString、equals 和 hashCode

該檢查會在以下 2 中情況下建議將 object 轉換為 data object：

  • 當自訂 toString 返回類別的名稱時
  • 當 object 繼承 sealed class/interface 時

範例：


    object Foo {
        override fun toString(): String = "Foo"
    }

在套用快速修復後：


    資料物件 Foo

僅當專案或模組的 Kotlin 語言級別為 1.9 或更高時，此檢查才會報告
