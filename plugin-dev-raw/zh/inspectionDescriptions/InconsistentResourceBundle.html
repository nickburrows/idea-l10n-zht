報告資源包中包含的屬性檔案中的問題。
  • 報告缺失的翻譯

    使用此選項可以報告在父屬性檔案中包含但在繼承屬性中缺失的屬性（除非它是語言方言）。

    範例：

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    # 空檔案
    空屬性 abc 將被報告為未翻譯。


  • 報告不一致的屬性

    使用此選項可以報告在繼承屬性檔案中包含但在父屬性（如果沒有父屬性，則在同級屬性）中缺失的屬性。

    範例：

    
    # messages.properties
    # 空檔案

    # messages_fr.properties
    abc=xxx
    此處的屬性 abc 翻譯不適用於法語以外的任何語言，因此，將在（預設）屬性檔案 messages.properties 中報告為缺失。


  • 報告使用相同值覆寫的屬性

    使用此選項可以報告逐字複製貼上到多個屬性檔案中的屬性。


    範例：

    
    # messages.properties
    abc=xxx

    # messages_fr.properties
    abc=xxx
    屬性 abc 將在檔案 messages_fr.properties 中報告為不必要地繼承。


  • 報告使用不同的佔位符覆寫的屬性

    使用此選項可以檢查被覆寫的屬性中的佔位符一致性。

    範例：

    
    # messages.properties
    qwe={0}xxx{1}
    abc={0}yyy{1}

    # messages_fr.properties
    qwe={0}xxx{0}xxx{1}
    abc={0}yyy
    屬性 abc 將被報告為包含訊息格式佔位符的屬性，與 messages.properties 不對應。


  • 報告使用不同值結尾覆寫的屬性

    使用此選項可以檢查被覆寫的屬性中的結尾一致性。

    範例：

    
    # messages.properties
    abc=xxxzzz

    # messages_fr.properties
    abc=xxx;
    屬性 abc 將被報告為以特殊符號 (! / ? / . / : / ;) 結尾，而 messages.properties 中的父值則不會。
