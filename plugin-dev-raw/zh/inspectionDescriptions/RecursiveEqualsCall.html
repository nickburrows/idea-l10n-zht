報告遞迴 equals(==) 呼叫。

在 Kotlin 中，== 會通過在背景呼叫 equals 方法來比較物件值。 另一方面，=== 則通過參照來比較物件。

=== 通常用於 equals 方法實作。 但 === 可能會錯誤地與 == 混淆，從而導致無限遞迴。

範例：


  class X {
      override fun equals(other: Any?): Boolean {
          if (this == other) return true
          return false
      }
  }

在套用快速修復後：


  class X {
      override fun equals(other: Any?): Boolean {
          if (this === other) return true
          return false
      }
  }
