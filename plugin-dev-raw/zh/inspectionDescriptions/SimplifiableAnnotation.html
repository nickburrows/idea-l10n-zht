報告可以簡化為單元素或標記速記形式的註解。

已報告的問題：

  • 註解名稱-值對中的冗餘 value=
  • 將僅包含單個值的陣列值括起來的冗餘大括號
  • @ 符號與註解名稱之間的冗餘空格
  • 註解名稱與參數列表之間的冗餘空格
  • 不帶任何參數的註解中的冗餘圓括號

範例：


  @interface Foo { String[] value(); }

  @ Foo({"foo"})
  public String name;

在套用快速修復後：


  @interface Foo { String[] value(); }

  @Foo("foo")
  public String name;
