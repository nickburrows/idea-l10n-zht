報告 abstract 類別的 public 建構函式。

abstract 類別的建構函式只能從其子類別的建構函式進行呼叫，將其宣告為 public 可能令人困惑。

該快速修復會將此類別建構函式設為 protected。

範例：


  public abstract class Foo {
    public Foo () { // 警告：有 'public' 修飾符
        /* ... */
    }
  }

在套用快速修復後：


  public abstract class Foo {
    protected Foo () {
        /* ... */
    }
  }

組態檢查：

使用下面的忽略非 public 類別選項可以忽略非 public 類別中的 public 建構函式。
