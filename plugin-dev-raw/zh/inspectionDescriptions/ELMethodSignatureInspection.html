報告 EL 中與 TLD 檔案中的 method-signature 約束不符合的方法呼叫。

範例 Java 類別：



public class ExampleAction {
  public void goodAction() { ... }
  public void badAction(String str) { ... }
}

範例 JSP 檔案：



<jsp:useBean id="example" scope="request" type="org.example.ExampleAction"/>
<f:view>
  <h:commandButton action="#{example.goodAction}"/> // 優良：方法接受零實參並返回 `void`
  <h:commandButton action="#{example.badAction}"/> // 錯誤：方法 `void badAction(String)` 返回 `java.lang.String` 類型而不是 `void`
</f:view>
