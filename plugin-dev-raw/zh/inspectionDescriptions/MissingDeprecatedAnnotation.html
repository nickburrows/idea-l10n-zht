報告具有 @deprecated Javadoc 標記但沒有 @java.lang.Deprecated 註解的模組宣告、類別、欄位或方法。

範例：


  /**
   * @deprecated use {@code example()} instead
   */
  void sample(){ }

在套用快速修復後：


  /**
   * @deprecated use {@code example()} instead
   */
  @Deprecated
  void sample(){ }

僅當專案或模組的語言級別為 5 或更高時，此檢查才會報告。

使用下面的複選框，報告使用 @Deprecated 註解但在 Javadoc @deprecated 標記中沒有解釋的成員。
