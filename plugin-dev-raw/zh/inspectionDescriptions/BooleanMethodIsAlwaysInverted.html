報告返回值類型為 boolean 且在調用時始終求反的方法。

提供了一種快速修復方法來反轉並可選擇地重新命名該方法。 由於性能原因，並不是所有有問題的方法都會在編輯器中醒目提示。

範例：


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

在套用快速修復後：


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
