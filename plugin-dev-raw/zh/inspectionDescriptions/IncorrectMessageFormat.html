報告不正確的訊息格式模式或佔位符的不正確索引

報告以下錯誤：

  • 未解析的索引或負索引
  • 未閉合的大括號
  • 不成對的引號。 在這種情況下，模式的一部分可能不會被使用
  • 可能引號的數量不正確
  • 嵌套選擇模式的下限不正確
  • 佔位符索引不正確。 在這種情況下，不能取代佔位符或不能使用實參

範例：


  MessageFormat.format("{wrong}", 1); // 不正確的索引
  MessageFormat.format("{0", 1); // 不符合的大括號
  MessageFormat.format("'{0}", 1); // 不成對的引號
  MessageFormat.format("It''''s {0}", 1); // 將列印“It''s”而非“It's”
  MessageFormat.format("{0}", 1, 2); // 模式中未使用索引為 '1' 的實參
