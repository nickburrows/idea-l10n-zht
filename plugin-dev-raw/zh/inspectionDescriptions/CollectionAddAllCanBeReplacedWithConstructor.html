在使用無參建構函式實例化集合後，立即報告 Collection.addAll() 和 Map.putAll() 呼叫。

此類別結構可被取代為對參數化建構函式的單次呼叫，從而簡化程式碼。 此外，對於某些集合，取代可能會更高效。

範例：


  Set<String> set = new HashSet<>();
  set.addAll(Arrays.asList("alpha", "beta", "gamma"));

在套用快速修復後：


  Set<String> set = new HashSet<>(Arrays.asList("alpha", "beta", "gamma"));

預設支援 JDK 集合類別。 此外，還可以使用要檢查的類別面板指定其他類別。
