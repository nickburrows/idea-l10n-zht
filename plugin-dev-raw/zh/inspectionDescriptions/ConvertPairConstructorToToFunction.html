報告可取代為 to() 中綴函式呼叫的 Pair 建構函式呼叫。

顯式建構函式呼叫可能會增添冗餘，尤其是在多次使用時。 將建構函式呼叫取代為 to() 會使讀取和維護程式碼變得更容易。

範例：


  val countries = mapOf(
      Pair("France", "Paris"),
      Pair("Spain", "Madrid"),
      Pair("Germany", "Berlin")
  )

在套用快速修復後：


  val countries = mapOf(
      "France" to "Paris",
      "Spain" to "Madrid",
      "Germany" to "Berlin"
  )
