報告不是所有抽象屬性或方法都在子類別中定義的情況。

範例：


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_triangle(self):
        pass

套用快速修復後，IDE 為 Triangle 類別實作抽象方法：


from abc import abstractmethod, ABC


class Figure(ABC):

    @abstractmethod
    def do_figure(self):
        pass


class Triangle(Figure):
    def do_figure(self):
        pass

    def do_triangle(self):
        pass
