報告使用 @com.intellij.openapi.components.Service 註解而不是 final 的類別。

如果某個類別為具體類別，則建議將該類設為 final。

範例：


  // MyService.kt
  @Service(Service.Level.APP)
  open class MyService

在套用快速修復後：


  // MyService.kt
  @Service(Service.Level.APP)
  class MyService

如果為抽象類別或接口，則建議移除 @Service 註解。

範例：


  // MyService.java
  @Service(Service.Level.APP)
  abstract class MyService {}

在套用快速修復後：


  // MyService.java
  abstract class MyService {}
