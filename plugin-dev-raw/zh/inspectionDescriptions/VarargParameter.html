報告接受任意數量參數的方法（也稱為 vararg 方法）。

範例：


  enum EnumConstants {
  A(null), B, C;

  EnumConstants(String... ss) {}
}

可通過快速修復將可變實參參數取代為等效的陣列參數。 方法呼叫中的相關實參包裝在陣列初始設定式表達式中。 在套用快速修復後：


  enum EnumConstants {
  A(null), B(new String[]{}), C(new String[]{});

  EnumConstants(String[] ss) {}
}

Varargs 方法在 Java 5 中出現。 此檢查有助於為向後相容早期 Java 版本而降級。
