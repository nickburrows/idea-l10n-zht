報告將 NULL 值插入僅接受 NOT NULL 值的列的情況。

範例（Microsoft SQL 伺服器）：

CREATE TABLE br2 (
id INT NOT NULL,
col1 NVARCHAR (20) NOT NULL,
col2 NVARCHAR (20) NOT NULL,
);
--
INSERT INTO br2 (id, col1, col2)
VALUES (1, NULL, NULL);

您不能在 col1 和 col2 中插入 NULL 值，因為它們被定義為 NOT NULL。 如果按原樣執行腳本，則會收到錯誤訊息。 要修復此程式碼，請將 VALUES 部分中的 NULL 取代為值（例如，42 和 'bird'）。

INSERT INTO br2 (id, col1, col2)
VALUES (1, 42, 'bird');
