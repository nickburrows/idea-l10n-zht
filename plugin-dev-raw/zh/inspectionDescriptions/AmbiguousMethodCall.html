如果在類別周圍的程式碼中存在具有相同簽名的方法，則報告從匿名類別、內部類別或區域類別對超類別方法的呼叫。 在這種情況下，看似呼叫了來自周圍程式碼的方法，而實際上呼叫的是來自超類別的方法。

為了闡明程式碼的意圖，建議對方法呼叫新增顯式的 super 限定符。

範例：


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        ambiguous(); //警告
      }
    }
  }

在套用快速修復後：


  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        super.ambiguous();
      }
    }
  }
