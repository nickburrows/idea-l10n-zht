報告對 super(A, B) 的任何呼叫不符合以下要求的情況：

  • B 是 A 的實例
  • B 是 A 的子類別

範例：


class Figure:
    def color(self):
        pass


class Rectangle(Figure):
    def color(self):
        pass


class Square(Figure):
    def color(self):
        return super(Rectangle, self).color() # Square 不是 Rectangle 的子類別

作為修復，您可以令 Square 為 Rectangle 類別的實例。
