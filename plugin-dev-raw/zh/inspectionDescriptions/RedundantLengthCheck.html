報告陣列迭代後不必要的陣列長度檢查。 當陣列長度為零時，迭代將被跳過，因此沒有必要顯式地檢查長度。

範例：


  void f(String[] array) {
    if (array.length != 0) { // 不必要的檢查
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

建議使用快速修復來解包或移除長度檢查：


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
