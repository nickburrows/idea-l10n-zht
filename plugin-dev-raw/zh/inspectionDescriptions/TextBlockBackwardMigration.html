報告可以取代為一般字串文字的文本塊。

範例：


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

在套用快速修復後：


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

文本塊在 Java 15 中出現。 此檢查有助於為向後相容早期 Java 版本而降級。
