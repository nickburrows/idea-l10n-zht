報告可以自動取代為增強型 switch 語句或表達式的 switch 語句。

範例：


  double getPrice(String fruit) {
    // Switch 語句可以取代為增強型 'switch'
    switch (fruit) {
      case "Apple":
        return 1.0;
      case "Orange":
        return 1.5;
      case "Mango":
        return 2.0;
      default:
        throw new IllegalArgumentException();
    }
  }

在套用快速修復後：


  double getPrice(String fruit) {
    return switch (fruit) {
      case "Apple" -> 1.0;
      case "Orange" -> 1.5;
      case "Mango" -> 2.0;
      default -> throw new IllegalArgumentException();
    };
  }

  • 使用僅在可以轉換為表達式時顯示警告選項，不對轉換為 switch 語句發出警告。
  • 使用一個分支中轉換為 switch 表達式的最大語句數選項，僅在每個分支的語句數小於給定的語句數時才對轉換為表達式發出警告。

僅當專案或模組的語言級別為 14 或更高時，此檢查才會報告。
