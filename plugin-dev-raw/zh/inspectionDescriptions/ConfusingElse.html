在 if—else 語句和語句鏈中報告冗餘的 else 關鍵字。

當所有之前的分支以 return、throw、break 或 continue 語句結尾時，else 關鍵字冗餘。 在這種情況下，來自 else 分支的語句可以放在 if 語句之後，並且 else 關鍵字可以刪除。

範例：


  if (name == null) {
      throw new IllegalArgumentException();
  } else {
      System.out.println(name);
  }

在套用快速修復後：


  if (name == null) {
      throw new IllegalArgumentException();
  }
  System.out.println(name);

停用當 'if' 語句之後沒有其他語句時報告選項，以忽略 if—else 語句是程式碼塊中最後一條語句的情況。
