檢測不正確的相依項表示法。

Gradle 支援以下類型/格式：
  • Dependency 的實例；
  • String 或 CharSequence 值，例如 'org.gradle:gradle-core:1.0'；
  • 映射，例如 [group: 'org.gradle', name: 'gradle-core', version: '1.0']；
  • FileCollection，例如 files('some.jar', 'someOther.jar')；
  • 專案，例如 project(':some:project:path')；
  • ClassPathNotation，例如 gradleApi()；
  • 相依項表示法列表，例如 ['org.gradle:gradle-core:1.0']；
  • (Gradle 7.0+) 版本目錄存取器，例如 libs.groovy.core。
請參閱 Gradle 文檔

範例：


dependencies {
  implementation(1) // reports '1'
}
