報告冗餘屬性 setter。

在以下情況之一中，setter 被認為是冗餘的：

  1. setter 沒有主體。 存取器可見性未更改，宣告不是 external 並且沒有註解。 
    
  var myPropWithRedundantSetter: Int = 0
      set // redundant

  var myPropA: Int = 0
      private set // 可以 - 屬性可見性被更改為 private

  var myPropB: Int = 0
      external set // 可以 - 未在 Kotlin 中實作（外部）

  var myPropC: Int = 0
      @Inject set // 可以 - 存取器被註解
  2. setter 主體是一個具有單語句的塊，其將參數分配給支援欄位。 
    
  var prop: Int = 0
      set(value) { // 冗餘
          field = value
      }
