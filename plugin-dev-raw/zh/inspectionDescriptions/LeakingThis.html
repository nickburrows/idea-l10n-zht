在物件建構期間使用 this 報告不安全的動作，包括：
  • 在類別初始化期間存取非 final 屬性：從建構函式或屬性初始化
  • 在類別初始化期間呼叫非 final 函式
  • 在非 final 類別的建構函式中使用 this 作為函式實參

如果其他類別繼承自給定的類別，則在執行不安全動作時，可能無法完全初始化它們。

範例：


  abstract class Base {
      val code = calculate()
      abstract fun calculate(): Int
  }

  class Derived(private val x: Int) : Base() {
      override fun calculate() = x
  }

  fun testIt() {
      println(Derived(42).code) // 預期值：42，實際值：0
  }
