報告可以簡化的求反布爾表達式。

該快速修復會簡化布爾表達式。

範例：


    fun test(n: Int) {
        !(0 == 1)
    }

在套用快速修復後：


    fun test(n: Int) {
        0 != 1
    }

請注意，如果涉及 IEEE-754 NaN 值，此動作可能會更改程式碼語意：


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

在套用快速修復後：


    fun main() {
        println(Double.NaN < 0) // false
    }
