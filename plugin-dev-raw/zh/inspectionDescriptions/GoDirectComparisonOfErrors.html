報告錯誤的直接比較，例如使用 ==，並建議改用 errors.Is。

從 Go 1.13 開始，可以使用 fmt.Errorf 函式和 %w 動詞來包裝錯誤。 因此，對於已包裝錯誤，使用相等檢查直接比較錯誤可能會失敗。 從 Go 1.13 開始，檢查特定錯誤的首選方式是使用標準庫中的 errors.Is 函式。

快速修復會將錯誤的直接比較取代為呼叫 errors.Is。

範例：


if err == MyError {
  fmt.Println("MyErr")
}

在套用快速修復後：


if errors.Is(err, MyError) {
  fmt.Println("MyErr")
}
