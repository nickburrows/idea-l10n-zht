报告可以简化的求反布尔表达式。

该快速修复会简化布尔表达式。

示例：


    fun test(n: Int) {
        !(0 == 1)
    }

在应用快速修复后：


    fun test(n: Int) {
        0 != 1
    }

请注意，如果涉及 IEEE-754 NaN 值，此操作可能会更改代码语义：


    fun main() {
        println(!(Double.NaN >= 0)) // true
    }

在应用快速修复后：


    fun main() {
        println(Double.NaN < 0) // false
    }
