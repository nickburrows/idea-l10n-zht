报告其中的 default 分支位于另一个 case 前面的 switch 语句或表达式。 此类构造会造成不必要的困惑。 如果可能，可以通过快速修复将 default 分支移到最后位置。

示例：


  switch (n) {
      default:
          System.out.println();
          break;
      case 1:
          break;
  }

在应用快速修复后：


  switch (n) {
    case 1:
        break;
    default:
        System.out.println();
        break;
  }
