报告可以替换为常规字符串文字的文本块。

示例：


  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

在应用快速修复后：


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

文本块在 Java 15 中出现。 此检查有助于为向后兼容早期 Java 版本而降级。
