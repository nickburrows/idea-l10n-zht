报告可以用 @FunctionalInterface 注解的接口（自 JDK 1.8 起提供）。

使用 @FunctionalInterface 对接口进行注解会指示该接口为函数式接口，不能再向其添加其他 abstract 方法。

示例：


  interface FileProcessor {
    void execute(File file);
  }

在应用快速修复后：


  @FunctionalInterface
  interface FileProcessor {
    void execute(File file);
  }

仅当项目或模块的语言级别为 8 或更高时，此检查才会报告。
