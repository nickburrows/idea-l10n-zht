报告返回值类型为 boolean 且在调用时始终求反的方法。

提供了一种快速修复方法来反转并可选择地重命名该方法。 由于性能原因，并不是所有有问题的方法都会在编辑器中高亮显示。

示例：


  class C {
    boolean alwaysTrue() {
      return true;
    }

    void f() {
      if (!alwaysTrue()) {
        return;
      }
    }
    boolean member = !alwaysTrue();
  }

在应用快速修复后：


  class C {
    boolean alwaysFalse() {
      return false;
    }

    void f() {
      if (alwaysFalse()) {
        return;
      }
    }
    boolean member = alwaysFalse();
  }
