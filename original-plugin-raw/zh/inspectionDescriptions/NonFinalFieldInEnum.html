报告枚举类型中的非 final 字段。 非 final 字段会引入全局可变状态，这通常会被认为不可取。

示例：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public String str;

    Enum(String str) {
        this.str = str;
    }
  }

在应用快速修复后：


  enum Enum {
    FIRST("first"),
    SECOND("second");

    public final String str;

    Enum(String str) {
        this.str = str;
    }
  }
使用“忽略不能设为 'final' 的字段”选项可以仅对可使用快速修复设为 final 的字段发出警告。
