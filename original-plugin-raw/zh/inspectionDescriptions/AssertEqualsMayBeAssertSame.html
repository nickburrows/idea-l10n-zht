报告可被替换为等效 assertSame() 调用的 JUnit assertEquals() 调用。 当实参是不重写 Object.equals() 方法并显式比较对象标识的 final 类的实例时，可能发生这种情况。

建议将 assertEquals() 替换为 assertSame()。

示例：


  @Test
  public void testObjectType() {
    Object o = getObject();
    Assert.assertEquals(String.class, o.getClass());
  }

在应用快速修复后：


  @Test
  public void testSort() {
    Object o = getObject();
    Assert.assertSame(String.class, o.getClass());
  }
