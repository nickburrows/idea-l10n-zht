报告可被替换为记录模式的模式。

示例：


  record Point(int x, int y) {}

  static void printSum(Object obj) {
    if (obj instanceof Point p) {
      int x = p.x();
      int y = p.y();
      System.out.println(x + y);
    }
  }

在应用快速修复后：


  record Point(int x, int y) {}

  static void printSum(Object obj) {
    if (obj instanceof Point(int x, int y)) {
      System.out.println(x + y);
    }
  }

仅当项目或模块的语言级别为 19 或更高时，此检查才会报告。
