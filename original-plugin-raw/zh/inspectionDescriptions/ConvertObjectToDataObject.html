报告可转换为 data object 的 object

data object 会自动生成 toString、equals 和 hashCode

该检查会在以下 2 中情况下建议将 object 转换为 data object：

  • 当自定义 toString 返回类的名称时
  • 当 object 继承 sealed class/interface 时

示例：


    object Foo {
        override fun toString(): String = "Foo"
    }

在应用快速修复后：


    数据对象 Foo

仅当项目或模块的 Kotlin 语言级别为 1.9 或更高时，此检查才会报告
