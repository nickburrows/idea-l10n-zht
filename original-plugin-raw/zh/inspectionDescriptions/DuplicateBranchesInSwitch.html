报告在不同分支中包含相同代码的 switch 语句或表达式，并建议合并重复的分支。

示例：


  switch (n) {
    case 1:
      System.out.println(n);
      break;
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }

在应用快速修复后：


  switch (n) {
    case 1:
    case 2:
      System.out.println(n);
      break;
    default:
      System.out.println("default");
  }
