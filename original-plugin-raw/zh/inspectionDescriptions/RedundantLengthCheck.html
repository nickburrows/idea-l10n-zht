报告数组迭代后不必要的数组长度检查。 当数组长度为零时，迭代将被跳过，因此没有必要显式地检查长度。

示例：


  void f(String[] array) {
    if (array.length != 0) { // 不必要的检查
      for (String str : array) {
        System.out.println(str);
      }
    }
  }

建议使用快速修复来解包或移除长度检查：


  void f(String[] array) {
    for (String str : array) {
      System.out.println(str);
    }
  }
