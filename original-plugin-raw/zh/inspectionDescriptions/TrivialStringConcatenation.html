报告其中一个实参为空字符串的字符串串联。 此类串联不必要。 有时，会作为习惯用法将非 String 对象或基元转换为 String，但在这种情况下，使用类似 String.valueOf 的方法会更清晰。

建议通过快速修复来简化串联。

示例：


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

在应用快速修复后：


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }

使用仅报告可以删除空字符串而不进行其他更改的情况选项仅报告可删除空字符串而不使用 String.valueOf 转换其他操作数的情况。
