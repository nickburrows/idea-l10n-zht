报告 SLF4J、Log4j2 和 akka.event.LoggingAdapter 日志记录调用，例如 logger.info("{}: {}", key)，其中记录器消息中 {} 占位符的数量与日志调用的其他实参的数量不匹配。

使用检查选项可以指定 SLF4J 使用哪个实现。 如果选择自动检查，则在类路径中搜索 org.apache.logging.slf4j.Log4jLogger。 如果找到了此文件或选择了是，将不为 SLFJ4 API 报告带有异常类型的最后一个形参具有占位符的情况。 示例： 


// 将不报告
log.error("For id {}: {}", id, new RuntimeException());
此选项仅适用于 SLF4J。
