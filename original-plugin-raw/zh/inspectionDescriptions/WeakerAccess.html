报告可能缩小其访问修饰符范围的字段、方法或类。

示例：


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    void bar(String x, String y) { } // 可以为 private
  }

在应用快速修复后：


  class Sample {
    void foo() {
        bar("foo", "foo");
    }
    private void bar(String x, String y) { }
  }

使用检查选项可以定义修饰符更改建议的规则。
